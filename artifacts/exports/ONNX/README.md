Я экспортировал обученную модель классификации пола (men/women) в формат TorchScript,
чтобы её можно было передавать и запускать как один файл без необходимости держать рядом исходный код модели.

В папке лежит файл cnn_flat8x8_torchscript.pt — это сама модель, и cnn_flat8x8_meta.json
— метаданные (размер входного изображения, соответствие индексов классам и прочие параметры).
Эта версия рассчитана на вход RGB 128×128.
